Evolve V1
Move blocks using arrow keys and combine them to form new characters.
Evolve V2
Similar to Evolve V1 in concept, but the canvas has a few walls which restrict the movement of blocks.
Quads
Juggle given numbers using basic arithmetic operations to arrive at the target number. Use arrow keys to navigate.
Bubbles
Chance-based game. Use arrow keys to keep the player under one of the circles all the time. The player has limited time outside the bubbles. Target is to reach a circle in one of the corners.
Color Swap
A game where you need to swap colors of the canvas to make a character visible in the foreground. It invokes recognizing morphological similarities between characters and using them to create a character.
Neighbors
My favorite game so far. I made this game as an attempt to simulate how human settlements take birth. You need to recognize the neighbors in the game based on a given criteria and then put them side by side in a way that maximizes the attractiveness of the 'settlement'.

Identify which blocks should be put together. Then click on the block in the right canvas and then click in a grid on the left canvas where you want to place the block. The aim is to gather as many points as possible by structuring the layout of the blocks.
Lying Pigs
A game where you need to guess the right character based on statements given by three pigs.
Itinerary
A game created for characters whose shapes have a common element. The player needs to pass through ever region in the canvas by changing its character. Use arrow keys to navigate.
Phonemes and radicals
A game to learn and practice the logic behind phonetics and radicals. You need to recognize the characters based on their meanings and pronunciation.

Try to get a picture of each character by clicking on the brown blocks and 'Hints' button. Then click the blocks on the right canvas and then click on the relevant block.
