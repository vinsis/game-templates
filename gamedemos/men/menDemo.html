Men2

Men2 means door. I didn't think much before naming this game.

Another easy to understand game for learning characters that follow a regular pattern.

  • The canvas is divided into different sections, each having a different color and a moving player. The moving player can't enter any of the inner blocks unless it changes the character it displays.
  • A player can change the character it displays by colliding into a relevant block.
  • Each section has a name (pinyin for a character). The player can enter a section only if the character it displays matches the pinyin of that section.

Aim:Your aim is to devise a path so as to cover all the sections of in the canvas. Once you have covered all the blocks, you will be greeted with a message.
